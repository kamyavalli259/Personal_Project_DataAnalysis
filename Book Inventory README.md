# Bookstore Inventory and Sales Analysis

**Duration:** Nov 2025 – Nov 2025  
**Tools Used:** SQL Server / MySQL, SQL  

## Project Overview
This project simulates a **Bookstore Management System** to track inventory, sales, and revenue. The goal is to demonstrate relational database design and data analysis using SQL.

Key Features:
- Maintain inventory of books (title, author, genre, price, stock).  
- Record sales transactions and analyze revenue.  
- Generate insights like bestselling books, genre-wise performance, total revenue, and low-stock alerts.

## Database Schema
The database contains two main tables:

- **Books** – stores book details and inventory information.  
- **Sales** – records sales transactions, linking to the `Books` table.
