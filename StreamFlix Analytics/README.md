## Project Overview
StreamFlix Analytics simulates a **movie streaming platform** tracking users, movies, views, ratings, and subscriptions. The goal is to analyze **user behavior, movie performance, and revenue** using SQL.  

Key Features:
- Track users, movies, views, ratings, and subscriptions.  
- Analyze most-watched movies and top-rated movies.  
- Demonstrate advanced SQL skills with joins, aggregations, and date formatting.

---

## Database Schema
- **Users:** user_id, name, email, signup_date  
- **Movies:** movie_id, title, genre, release_year, duration_min  
- **Views:** view_id, user_id, movie_id, view_date, duration_watched  
- **Ratings:** rating_id, user_id, movie_id, rating, rating_date  
- **Subscriptions:** subscription_id, user_id, start_date, end_date, price
